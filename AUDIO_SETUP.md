# 音频设备安装指南

## Windows 用户

### 方案1: 使用虚拟音频线 (推荐)

1. 下载 VB-Cable Virtual Audio Device:
   https://vb-audio.com/Cable/

2. 安装后重启电脑

3. 在软件中选择输出设备为 "CABLE Input"

4. 在直播软件（OBS/抖音直播伴侣）中选择麦克风为 "CABLE Output"

### 方案2: 使用立体声混音

1. 右键任务栏音量图标 → 声音
2. 录制选项卡 → 右键空白处 → 显示禁用的设备
3. 启用 "立体声混音"
4. 右键 "立体声混音" → 属性 → 侦听 → 勾选 "侦听此设备"
5. 选择播放设备为你的输出设备

## macOS 用户

使用 BlackHole 虚拟音频设备:
https://existential.audio/blackhole/

## 音频路由示意图

```
麦克风 → [本软件] → 虚拟音频线 → 直播软件 → 观众
            ↓
        处理后播放
```

## 常见问题

Q: 为什么听不到声音？
A: 软件设计为输出到虚拟设备，不会从扬声器播放。请检查直播软件是否选择了正确的输入设备。

Q: 如何测试是否正常工作？
A: 在直播软件的音频预览中查看是否有波形跳动。

Q: 有延迟怎么办？
A: 虚拟音频设备通常有 10-30ms 延迟，属正常范围。如延迟过大，尝试降低缓冲区设置。
