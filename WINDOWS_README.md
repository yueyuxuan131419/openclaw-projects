# 青丘主播替身 - Windows 使用指南

## 📦 文件说明

打包后会生成以下文件：
```
dist/
├── 青丘主播替身.exe    # 主程序
├── ...                # 依赖文件
```

## 🚀 使用方法

### 方法1：直接运行（推荐）
1. 将整个 `dist` 文件夹复制到任意位置
2. 双击运行 `青丘主播替身.exe`
3. 首次运行会请求麦克风权限，请点击 **"允许"**

### 方法2：创建快捷方式
1. 右键 `青丘主播替身.exe`
2. 选择 **"发送到" → "桌面快捷方式"**
3. 以后从桌面双击运行

## 🔧 Windows 音频设置

### 步骤1：安装虚拟音频线（推荐 VB-Cable）
1. 下载 VB-Cable：https://vb-audio.com/Cable/
2. 解压，以管理员身份运行 `VBCABLE_Setup.exe`
3. 安装完成后重启电脑

### 步骤2：配置直播软件
1. 打开直播伴侣 / OBS
2. 设置 → 音频 → 麦克风
3. 选择 **"CABLE Output (VB-Audio Virtual Cable)"**

### 步骤3：配置本软件
1. 运行 `青丘主播替身.exe`
2. 麦克风：选择你的真实麦克风
3. 输出到：选择 **"CABLE Input (VB-Audio Virtual Cable)"**

## 🎮 使用流程

```
┌─────────────────────────────────────┐
│  1. 点击 🎤 直播 开始录制           │
│     （对着麦克风说话）               │
├─────────────────────────────────────┤
│  2. 需要回答评论时：                 │
│     点击 💬 互动（暂停录制）         │
│     回答完再点 🎤 直播 继续          │
├─────────────────────────────────────┤
│  3. 累了想休息时：                   │
│     点击 🤖 替身                     │
│     软件会循环播放已录制的内容       │
│     ⏱️ 倒计时提醒你剩余时间          │
├─────────────────────────────────────┤
│  4. 休息好了：                       │
│     点击 🎤 直播 恢复录制            │
│     （替身内容会被清空，重新开始）   │
└─────────────────────────────────────┘
```

## ⚠️ 注意事项

1. **管理员权限**：如果无法正常运行，尝试右键 → "以管理员身份运行"

2. **杀毒软件**：部分杀毒软件可能误报，请将程序添加到白名单

3. **音频设备**：
   - 如果找不到设备，点击"刷新设备"按钮
   - 确保麦克风和虚拟音频线已正确安装

4. **首次运行**：
   - Windows 会弹出防火墙提示，请点击"允许"
   - 会请求麦克风权限，请点击"允许"

## 🔴 常见问题

### Q1: 提示 "无法启动音频录制"
- 检查麦克风是否被其他程序占用
- 尝试重启电脑
- 检查麦克风权限是否已开启

### Q2: 替身模式没有声音
- 检查输出设备是否正确设置为虚拟音频线
- 检查直播软件是否选择了正确的输入设备
- 检查系统音量是否静音

### Q3: 打包后的 exe 被杀毒软件拦截
- 这是正常现象，Python 打包的程序常被误报
- 将程序添加到杀毒软件白名单
- 或暂时关闭杀毒软件

### Q4: 界面显示乱码
- 确保系统安装了中文字体
- 将系统区域设置为中文（控制面板 → 区域 → 管理 → 更改系统区域设置 → 中文）
